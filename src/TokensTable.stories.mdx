## Tokens table

Displays a table with the (JSON) path of the token, the current value (which can be
edited) and where the current value is coming from (either a reference to another
token, or a literal value).

export const Template = ({container, limitTo=null}) => (
  <TokensTable container={container} limitTo={limitTo} />
);

## Token row

export const Template = ({designToken, contained = false}) => (
  <div className={contained ? 'dte-tokens-block' : ''}>
    <TokenRow designToken={designToken} />
  </div>
);

The token row adapts its layout based on the container it is in - by default it's
displayed in 'mobile'/'narrow' view, but on wider containers the content is displayed in
a single row.

_This relies on the `@container` queries, which are fairly new - not all browsers
support this yet!_
